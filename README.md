### 1. 열화상 카메라로 타이어 온도 시각화(메인 플젝)
열화상 카메라 데이터는 CAN 버스로 전송하기에는 용량이 너무 크다.
따라서 STM32로 데이터 전처리를 해야 할듯하다.
(타이어 엣지 온도만 본다던지 평균값을 본다던지.. 또는 데이터를 직렬?로 바꿔서 CAN버스에 집어넣던지 이건 생각을 좀 해봐야할듯)

#### 구상
STM + CAN 버스 모듈 + 열화상 카메라(데이터 전처리) -> CAN버스 전송 -> 라즈베리파이 데이터로거에서 수신 -> 전처리된 데이터 시각화
